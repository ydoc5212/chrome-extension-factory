to Chromium Extensions, Patrick Kettner, Oliver Dunk

Hi Chromium Extensions team,

We recently submitted an update to the Chrome Web Store that addresses a breaking change in Chromium 144’s messaging API which causes user login failures if the extension is not updated.

The update does not introduce any new permissions and only adapts existing messaging logic to maintain compatibility with Chromium 144.

This extension has ~2 million active users, so the Chromium 144 rollout (scheduled for Jan 13 according to the PSA announcement) could result in user-facing login issues if the update does not clear review in time.

Chrome Web Store item ID: **hnmpcagpplmpfojmgmnngilcnanddlhb**

Submission date: Jan 11

If possible, we wanted to ask whether this submission could be reviewed on an expedited basis. Please let me know if any additional information is needed.

Thanks for your time.
